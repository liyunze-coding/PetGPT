# PetGPT

This is a UI clone of [ChatGPT](https://chat.openai.com) by OpenAI, inspired by [CatGPT](https://www.cat-gpt.com). This app is built on Sveltekit, Typescript and TailwindCSS.

## Development

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run build
```
